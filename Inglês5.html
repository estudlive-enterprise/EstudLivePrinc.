THERE IS / THERE ARE

O que são?

"There is" e "There are" são estruturas usadas para indicar a existência de algo em um determinado lugar ou situação. A diferença principal está no número do substantivo que acompanha a expressão:

  • There is → singular ou incontável
  • There are → plural

Estrutura Básica

Forma afirmativa: There is / There are + substantivo + complemento
Forma negativa: There isn’t / There aren’t + substantivo
Pergunta: Is there / Are there + substantivo?

Exemplos de There is

  • There is a book on the table. (Há um livro sobre a mesa.)
  • There is milk in the fridge. (Há leite na geladeira.)
  • There isn’t any sugar. (Não há açúcar.)
  • Is there a problem? (Há algum problema?)

Exemplos de There are

  • There are three cats in the garden. (Há três gatos no jardim.)
  • There are many students in the class. (Há muitos alunos na sala.)
  • There aren’t any chairs here. (Não há cadeiras aqui.)
  • Are there any tickets left? (Ainda há ingressos?)

Uso com quantifiers

É comum combinar there is / there are com quantifiers como some, any, many, much, a lot of:

  • There is some water in the bottle. (Há um pouco de água na garrafa.)
  • There aren’t any apples. (Não há maçãs.)
  • There are many books on the shelf. (Há muitos livros na estante.)

Dicas avançadas

  • Para situações futuras, use there will be: There will be a meeting tomorrow. (Haverá uma reunião amanhã.)
  • Para enfatizar existência negativa: there is no / there are no – There is no reason to worry. (Não há motivo para se preocupar.)
  • Em perguntas polidas, Is there any…? ou Are there any…? é mais natural: Are there any questions? (Há alguma pergunta?)
  • Evite confundir there is com it is: There is a problem indica existência; It is a problem descreve ou identifica algo.

Resumo Final

"There is / There are" é essencial para indicar a existência de coisas, pessoas ou situações. Combinando com quantifiers e advérbios, você consegue expressar quantidade e localização com precisão.

Dica extra: pratique descrevendo cômodos, fotos, ou cenários do dia a dia usando there is / there are para internalizar o uso correto em contextos reais.
