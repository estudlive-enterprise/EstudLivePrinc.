Matemática – Funções

Funções

O que é uma Função?

Em matemática, uma função é uma regra que associa cada elemento de um conjunto inicial a um único elemento de outro conjunto.

Em outras palavras: para cada valor de x, existe apenas um valor correspondente de y.

Domínio, Contradomínio e Imagem

Toda função é formada por três partes fundamentais:

Domínio

É o conjunto de todos os valores que podem ser atribuídos à variável independente (geralmente o x).

Contradomínio

É o conjunto que contém todos os valores que podem ser resultado da função.

Imagem

É o conjunto formado pelos valores que realmente aparecem como resultado da função.

Representação de uma Função

Uma função pode ser representada de várias formas:

  • Por fórmula matemática: f(x) = 2x + 1
  • Por tabela de valores
  • Por gráfico no plano cartesiano
  • Por diagrama de setas

Função Afim (Função do 1º Grau)

A função afim é a função mais importante do 1º ano do Ensino Médio.

Sua forma geral é:
f(x) = ax + b

  • a → coeficiente angular (inclinação da reta)
  • b → coeficiente linear (onde a reta corta o eixo y)

Coeficiente Angular (a)

O coeficiente angular indica se a função é:

  • Crescente: a > 0
  • Decrescente: a < 0
  • Constante: a = 0

Gráfico da Função Afim

O gráfico de uma função afim é sempre uma reta.

Para construir o gráfico:

  1. Escolha dois valores para x
  2. Calcule os valores de y
  3. Marque os pontos no plano cartesiano
  4. Ligue os pontos com uma reta

Importância das Funções

Funções são usadas para modelar situações do cotidiano, como crescimento populacional, gastos, velocidade, temperatura e economia.

Elas são a base para conteúdos mais avançados como função quadrática, exponencial e logarítmica.
