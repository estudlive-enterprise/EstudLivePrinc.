Comandos Básicos de Python

Introdução

Python é uma linguagem de programação de alto nível, interpretada e de fácil leitura. É amplamente utilizada para desenvolvimento web, ciência de dados, inteligência artificial e muito mais.

Sintaxe

A sintaxe do Python é simples e direta. Por exemplo, para imprimir algo na tela, você pode usar:

print("Olá, Mundo!")

Variáveis

Em Python, você pode criar variáveis simplesmente atribuindo valores a elas:

nome = "João"
idade = 25

Condicionais

Use estruturas condicionais para tomar decisões no código:

if idade >= 18:
  print("Você é maior de idade")
else:
  print("Você é menor de idade")

Loops

Os loops permitem repetir blocos de código. Por exemplo, um loop for:

for i in range(5):
  print(i)
