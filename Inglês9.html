FALSE COGNATES

O que são False Cognates?

False cognates são palavras em inglês que se parecem com palavras em português, mas possuem significado diferente. Essas palavras podem gerar confusão se forem interpretadas literalmente pelo português.

Exemplos comuns

  • Actual ≠ atual → Actual significa real, verdadeiro. Ex.: The actual situation is different. (A situação real é diferente.)
  • Library ≠ livraria → Library significa biblioteca. Ex.: I borrowed a book from the library. (Peguei um livro na biblioteca.)
  • Pretend ≠ pretender → Pretend significa fingir. Ex.: The child pretended to be a superhero. (A criança fingiu ser um super-herói.)
  • Push ≠ puxar → Push significa empurrar. Ex.: Push the door to open it. (Empurre a porta para abri-la.)
  • Assist ≠ assistir → Assist significa ajudar. Ex.: She assisted me with the homework. (Ela me ajudou com o dever de casa.)

Dicas para evitar erros

  • Quando uma palavra parecer familiar, sempre confira o contexto antes de traduzir.
  • Crie uma lista pessoal de false cognates que você encontra com frequência.
  • Use exemplos em frases reais para fixar o significado correto.
  • Leia textos variados e sublinhe palavras que parecem conhecidas, mas têm significado diferente.
  • Pratique com exercícios de múltipla escolha que exigem identificar o sentido correto.

Exercício prático

Escolha a tradução correta:

  1. The actual cost of the project is high. → a) atual b) real
  2. I went to the library to study. → a) livraria b) biblioteca
  3. He pretended to be asleep. → a) pretendia b) fingia

Respostas: 1-b, 2-b, 3-b

Resumo Final

Aprender false cognates é essencial para evitar traduções literais incorretas. Reconhecer essas palavras e praticar seu uso correto melhora leitura, escrita e compreensão oral.

Dica extra: sempre que encontrar uma palavra parecida com português, pergunte-se: “Será que o significado real é o mesmo?” e confirme com dicionário confiável ou contexto do texto.
